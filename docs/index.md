# Welcome to ADAM!

This online manual is designed to lead system administrators and users through the most common aspects of managing an ADAM database. This guide should also enable administrators to provide their users with competent user support.

## Additional Support

Any issues that are not covered in this help documentation can be addressed to [help@adam.co.za](mailto:help@adam.co.za) for further information.

## Suggesting Changes

This is a living document and will be added to and improved over time. If you have any suggestions or wish to assist in improving the documentation, you can contribute through [GitHub](https://github.com/phinor/adam-docs).
